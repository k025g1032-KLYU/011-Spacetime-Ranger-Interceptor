# Spacetime Ranger

## Overview

Spacetime Ranger は、隊員を配置し、敵を包囲・確保する戦略パズルゲームです。

隊員ごとに移動速度・体格・索敵範囲が異なり、索敵範囲は時間経過によって徐々に拡大します。

敵が索敵範囲へ侵入すると、敵は即座に別の座標へ転移します。この転移ではゲーム内時間は経過しません。

プレイヤーは複数の隊員を配置し、転移を繰り返す敵を少しずつ包囲し、最後は隊員自身が体当たりして敵を確保します。

---

## Features

### 時間停止を利用した連鎖システム

敵が索敵範囲へ侵入すると瞬時に転移します。

転移中はゲーム内時間が停止するため、一度の時間経過の中で複数の隊員による連鎖的な迎撃を発生させることができます。

---

### 隊員ごとの個性

各隊員には

- 移動速度
- 体格
- 索敵範囲

が設定されています。

状況に応じて適切な隊員を配置することが重要になります。

---

### 成長する索敵範囲

索敵範囲は時間とともに広がります。

待機時間を利用することで広範囲をカバーできますが、敵の移動タイミングも考慮する必要があります。

---

### 包囲システム

敵を一度捕まえるだけではクリアになりません。

複数回の転移を利用して逃げ道を塞ぎ、最終的に隊員が接触することで逮捕となります。
